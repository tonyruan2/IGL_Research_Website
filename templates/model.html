User Guide

1) Input valid numbers for the process probability [0 to 1, with increments of 0.001], desired probability [A and B are integers], and max trials [10 to 1000]. Larger max trial values may cause model generation to take slightly more time.

2) Select to highlight monotonicity or periodicity if desired.

3) Select to draw helper lines if desired.

4) Click the create button to create a model.

5) Use the tools provided to [1] pan the graph, [2] zoom in and out, [3] reset the graph to its default state, [4] save the graph, and [5] hover over data points.

Model Information

This model shows the probability of achieving at least the desired probability with a certain trial count.

When monotonicity is selected, the points on the graph are colored according to their trial count mod B, where A divided by B is the desired probability in simplest terms. Points with trial counts that are equivalent mod B have the same color. Preset colors are used for the first 8 groups of values equivalent mod B. Afterwards, random colors are generated for each group of values equivalent mod B.

When periodicity is selected, every B trials (starting from and implicitly including 0) are colored the same color. Similar to the highlighting when monotonicity is selected, the first eight groups have the same color. Afterwards, random colors are generated.
